# permissions (Nested-Attribute)

This represents all permissions for an Administrator, which can be used to manage access control and capabilities within the system.





## Nested Attributes
- `permission` (List of Blocks, see [here](attr_admin_permission.md)) A set of permissions granted to the administrator, each containing details about the permission.
