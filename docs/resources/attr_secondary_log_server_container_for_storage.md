# secondary_log_server_ref (Nested-Attribute)

This represents the Secondary Log Server Container for Storage, which includes a reference to a Log Server and its rank in the configuration.




## Simple Attributes
- `rank` (Number) The rank of the secondary log server, which determines its order in the configuration.
- `value` (String) This represents a Log Server, which is a component of the Management Center responsible for storing and managing log (and alert) data, and analyzing and correlating events detected by multiple NGFW Engines.
