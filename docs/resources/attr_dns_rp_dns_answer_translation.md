# dns_answer_translation (Nested-Attribute)

This represents the definition of DNS Answer translations, which includes an original IP address or network and a translated IP address or list of hostnames.




## Simple Attributes
- `original_ipaddress` (String) The original IP address or network that is being translated in DNS answer translation.
- `translated_ipaddress` (String) The translated IP address or list of hostnames that corresponds to the original IP address in DNS answer translation.
