# move_conflict (Nested-Attribute)

This represents a Move Conflict generated in case of conflict during the Move process




## Simple Attributes
- `authorized_method_list` (List of String) The conflict allowed methods.
- `default_method` (String) The conflict resolution method.
- `element` (String) This is a base class for objects that can be stored in the database. It includes properties such as key, comment, and links.
