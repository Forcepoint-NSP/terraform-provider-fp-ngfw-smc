# domain_specific_dns_server (Nested-Attribute)

This represents the definition of Domain specific DNS Servers, which includes the domain name and associated DNS server addresses.




## Simple Attributes
- `dns_server_addresses` (String) The DNS server addresses associated with the domain, which can be a single IP address or a list of IP addresses separated by commas.
- `domain_name` (String) The domain name for which the DNS server addresses are defined.
