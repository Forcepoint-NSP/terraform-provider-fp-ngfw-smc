# sm_rp_candidate_entry (Nested-Attribute)

This represents the PIM-SM Bootstrap Settings RP Candidate entry for a Firewall, which includes the Multicast Group and optional manual Multicast IPv4 Network.




## Simple Attributes
- `multicast_ip_network` (String) The manual Multicast IPv4 Network associated with this RP Candidate entry, if available.
- `rp_multicast_group_ref` (String) This represents a Network, which is a Network Element that represents a (sub)network of computers. It includes attributes for IPv4 and IPv6 networks, broadcast address, and validation patterns.
