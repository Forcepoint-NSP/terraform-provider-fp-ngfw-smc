# boolean_value (Nested-Attribute)

This represents the Protocol Agent Boolean parameter value, which can be either '0' for false or '1' for true.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The boolean value, where '0' represents false and '1' represents true.
