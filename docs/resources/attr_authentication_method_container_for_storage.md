# authentication_server (Nested-Attribute)

This represents an authentication method container used for storage, which includes the rank of the authentication server and a reference to the authentication server itself.




## Simple Attributes
- `auth_server_ref` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
- `rank` (Number) The rank of the authentication server in the current authentication method.
