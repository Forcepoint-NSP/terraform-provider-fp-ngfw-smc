# custom_property (Nested-Attribute)

This represents a Custom Property, which is a key-value pair used in various configurations. It can be a regular string or an encrypted value.




## Simple Attributes
- `data_type` (String) The data type of the custom property, which can be 'string' for regular strings or 'encrypted' for encrypted values.
- `encrypted_value` (String) The encrypted value of the custom property, used for sensitive information such as passwords.
- `name` (String) The name of the custom property, which serves as the key in the key-value pair.
- `value` (String) The value of the custom property, which can be a regular string or an encrypted value.
