# as_number (Nested-Attribute)

This represents the AS Number element for a Route Entry Settings in a Route Map Rule, including the rank and expression value.




## Simple Attributes
- `expression` (String) The expression value used in the Route Entry Settings.
- `rank` (Number) The rank of this expression, determining its order of evaluation.
