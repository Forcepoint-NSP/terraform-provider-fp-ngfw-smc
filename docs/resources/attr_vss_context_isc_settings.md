# vc_isc (Nested-Attribute)

This represents the ISC settings for a VSS Context, including the traffic tag, policy ID, and ISC name.




## Simple Attributes
- `isc_policy_id` (Number) The policy ID associated with the VSS Context ISC Settings, which is required for the configuration.
