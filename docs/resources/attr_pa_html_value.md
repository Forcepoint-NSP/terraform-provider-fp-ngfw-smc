# html_value (Nested-Attribute)

This represents the Protocol Agent Html parameter value, which can be used to configure HTML content in the Protocol Agent.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The HTML content used in the Protocol Agent Parameter.
