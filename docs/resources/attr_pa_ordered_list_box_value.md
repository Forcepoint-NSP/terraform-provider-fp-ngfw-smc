# ordered_list_box_value (Nested-Attribute)

This represents the Protocol Agent Ordered List Box parameter value, which is a string of selected ordered list box rows in the format 'X,Y,...' where X, Y are the indices of the selected rows.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The Ordered List Box value, represented as a comma-separated string of selected indices.
