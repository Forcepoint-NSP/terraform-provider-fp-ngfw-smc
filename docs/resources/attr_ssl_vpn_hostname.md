# ssl_vpn_hostname (Nested-Attribute)

This represents the SSL VPN hostname, which can be an IP Address or a Fully Qualified Domain Name (FQDN).




## Simple Attributes
- `ssl_vpn_hostname` (String) The hostname for the SSL VPN, which can be an IP Address or a Fully Qualified Domain Name (FQDN).
