# ospf_abr_substitute_container (Nested-Attribute)

This represents the OSPF ABR Substitute container elements, which include a subnet reference, an ABR substitute type, and an optional substitute reference.




## Simple Attributes
- `subnet_ref` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
- `substitute_ref` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
- `substitute_type` (String) The type of OSPF ABR substitute. Required for the OSPF ABR substitute container.
