# capabilities (Nested-Attribute)

This represents the capabilities of a gateway or VPN profile, including various cryptographic and protocol features.




## Simple Attributes
- `sha2_ike_hash_length` (Number) The SHA2 hash length used for IKE (Internet Key Exchange).
- `sha2_ipsec_hash_length` (Number) The SHA2 hash length used for IPsec (Internet Protocol Security).
