# fixed_domain_answer (Nested-Attribute)

This represents a fixed domain answer in the DNS Relay Profile, which includes a domain name, a translated domain name, and a translated IP address.




## Simple Attributes
- `domain_name` (String) The domain name for the fixed domain answer.
- `ipaddress` (String) The translated IP address for the fixed domain answer.
- `translated_domain_name` (String) The translated domain name for the fixed domain answer.
