# distance_entry (Nested-Attribute)

This represents the BGP distance entry, which includes the subnet and distance value.




## Simple Attributes
- `distance` (Number) The distance value.
- `subnet` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
