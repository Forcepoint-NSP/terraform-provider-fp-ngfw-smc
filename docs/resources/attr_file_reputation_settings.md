# file_reputation_settings (Nested-Attribute)

This represents the File Reputation settings for the Engine Cluster, including the file reputation context and HTTP proxies.




## Simple Attributes
- `file_reputation_context` (String) The file reputation context, which is required for the File Reputation Settings.
- `http_proxy` (List of String) URI of the HTTP Proxy.
