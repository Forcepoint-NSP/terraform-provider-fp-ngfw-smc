# authentication_options (Nested-Attribute)

This represents the Firewall User Authentication Options, which is a sub part of Firewall Access Rule used to describe the User Authentication part of the rule.




## Simple Attributes
- `methods` (List of String) URI of the authentication method.
- `require_auth` (Boolean) Indicates whether the users are required to authenticate themselves.
- `users` (List of String) URI of the user or user group.
