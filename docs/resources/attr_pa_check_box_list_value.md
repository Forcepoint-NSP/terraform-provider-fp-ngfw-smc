# checkbox_list_value (Nested-Attribute)

This represents the Protocol Agent CheckBox List parameter value, which is a string of selected check-box rows in the format 'X,Y,...' where X, Y are the indices of the selected check-boxes.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The CheckBox List value, represented as a comma-separated string of selected indices.
