# ssh_profile_value (Nested-Attribute)

This represents the Protocol Agent SSH Profile parameter value, which is used to reference an SSH profile in the Protocol Agent configuration.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `ssh_profile_ref` (String) This represents an Ssh Profile, which contains lists of key exchange algorithms, cipher algorithms, and message authentication code algorithms used in SSH connections.
