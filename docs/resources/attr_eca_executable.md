# eca_executable (Nested-Attribute)

This represents an ECA Executable, which contains metadata about an executable file used in ECA Custom situations. It includes fields such as product name, file name, version number, MD5 hash, and SHA256 hash.




## Simple Attributes
- `file_name` (String) The name of the executable file.
- `md5_hash` (String) The MD5 hash of the executable file, used for integrity verification.
- `product_name` (String) The name of the product associated with the executable.
- `sha256_hash` (String) The SHA256 hash of the executable file, used for integrity verification.
- `version_number` (String) The version number of the executable file.
