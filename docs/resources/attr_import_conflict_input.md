# import_conflict (Nested-Attribute)

This represents an import conflict generated in case of conflict during the import process




## Simple Attributes
- `authorized_method_list` (List of String) The list of authorized methods to resolve conflict.
- `conflict_type` (String) The type of conflict.
- `default_method` (String) The default import conflict method.
- `element` (String) This is a base class for objects that can be stored in the database. It includes properties such as key, comment, and links.
