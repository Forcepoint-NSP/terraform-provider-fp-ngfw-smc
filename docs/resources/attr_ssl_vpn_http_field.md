# ssl_vpn_http_field (Nested-Attribute)

This represents an HTTP field to be used in an SSL VPN, which includes the field name and value.




## Simple Attributes
- `http_field_name` (String) The name of the HTTP field, which is used to identify the field in HTTP requests.
- `http_field_value` (String) The value of the HTTP field, which is used to set the field in HTTP requests.
