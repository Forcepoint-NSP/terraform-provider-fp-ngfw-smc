# ssm_advanced_setting (Nested-Attribute)

This represents the advanced settings for Sidewinder Proxies, including proxy type, attribute, and value.




## Simple Attributes
- `attribute` (String) The attribute or name of the setting for the Sidewinder Proxy.
- `type` (String) The type of Sidewinder Proxy. Allowed values are 'shared', 'ftp', 'http', 'smtp', 'ssh', 'tcp', and 'udp'.
- `value` (String) The value of the setting for the Sidewinder Proxy.
