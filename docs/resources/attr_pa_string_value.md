# string_value (Nested-Attribute)

This represents the Protocol Agent String parameter value, which can be used to configure string-based parameters in the Protocol Agent.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The string value used in the Protocol Agent Parameter.
