# gateway_nodes_usage (Nested-Attribute)

This represents an entry with all possible gateway nodes used by an Internal Gateway inside a VPN, including the satellite, central, and mobile gateway nodes.




## Simple Attributes
- `central_gateway_node_ref` (String) This represents a gateway node, which is used to manage the VPN topology and its nodes, including their usage and relationships.
- `mobile_gateway_node_ref` (String) This represents a gateway node, which is used to manage the VPN topology and its nodes, including their usage and relationships.
