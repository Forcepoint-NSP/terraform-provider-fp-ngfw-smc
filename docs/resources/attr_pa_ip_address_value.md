# ip_address_value (Nested-Attribute)

This represents the Protocol Agent IP Address parameter value, which can be used to configure IP address-based parameters in the Protocol Agent.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The IP address value used in the Protocol Agent Parameter.
