# integer_value (Nested-Attribute)

This represents the Protocol Agent Integer parameter value, which can be used to configure various integer-based parameters in the Protocol Agent.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (Number) The integer value used in the Protocol Agent Parameter.
