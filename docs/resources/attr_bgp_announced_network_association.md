# announced_ne_setting (Nested-Attribute)

This represents the announced Network Element entry for BGP Settings, including the Network Element and an optional RouteMap Policy to apply.




## Simple Attributes
- `announced_ne_ref` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
- `announced_rm_ref` (String) This represents a Route Map Policy for the Dynamic Routing Firewall settings, which is used to control the routing behavior based on specific rules.
