# checkbox_value (Nested-Attribute)

This represents the Protocol Agent Check Box parameter value, which can be either '0' for unselected or '1' for selected.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (String) The check-box value, where '0' represents unselected and '1' represents selected.
