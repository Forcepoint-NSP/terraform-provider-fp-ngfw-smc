# macro_value (Nested-Attribute)

This represents the Protocol Agent Macro parameter value, which can be either '1' for $CLUSTER_IPS or '0' for $NODE_IP.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.
- `value` (Number) The macro value, where '1' represents $CLUSTER_IPS and '0' represents $NODE_IP.
