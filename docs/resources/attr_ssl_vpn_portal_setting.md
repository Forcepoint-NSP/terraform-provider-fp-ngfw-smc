# ssl_vpn_portal_setting (Nested-Attribute)

This represents the Application Access Portal Setting, which includes the Application Access Portal element and its service port.




## Simple Attributes
- `port` (Number) The SSL VPN Portal service port, which defines the port used for the SSL VPN Portal.
- `ssl_vpn_portal` (String) This represents an Application Access Portal. It contains settings for hostnames, policies, server credentials, and various portal configurations.
