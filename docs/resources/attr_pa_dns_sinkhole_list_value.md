# pa_dns_sinkhole_list (Nested-Attribute)

This represents the Protocol Agent DNS Sinkhole List parameter value, which contains a list of DNS Sinkhole rules.




## Simple Attributes
- `parameter_ref` (String) This represents a parameter for the Protocol Agent, allowing for detailed configuration of agent settings.

## Nested Attributes
- `dns_sinkhole_rules` (List of Blocks, see [here](attr_sinkhole_protocol_parameter_value.md)) The DNS Sinkhole rule used in the Protocol Agent Parameter.
