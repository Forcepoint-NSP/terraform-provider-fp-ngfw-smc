# ranked_url (Nested-Attribute)

This represents an ordered URL, which can be used to prioritize URLs in a list, such as for updates or configurations.




## Simple Attributes
- `rank` (Number) The rank of the URL in the list, indicating its order or priority.
- `url` (String) The URL associated with this ranked entry.
