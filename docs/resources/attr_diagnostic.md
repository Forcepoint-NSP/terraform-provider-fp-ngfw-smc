# diagnostic (Nested-Attribute)

This represents a diagnostic, which is used to enable or disable diagnostic logging.




## Simple Attributes
- `enabled` (Boolean) Indicates whether the diagnostic is enabled or not.
- `name` (String) Name of the object.
