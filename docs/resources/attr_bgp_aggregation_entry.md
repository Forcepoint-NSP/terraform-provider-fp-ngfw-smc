# aggregation_entry (Nested-Attribute)

This represents the BGP aggregation entry, which includes the subnet and aggregation mode.




## Simple Attributes
- `mode` (String) The aggregation mode, which can be one of the following: aggregate, aggregate_as_set, summary_only, or as_set_and_summary.
- `subnet` (String) This represents a network element, which is a component that has an IP address and can be part of a network. It includes a location reference.
